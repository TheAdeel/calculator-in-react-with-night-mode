# Calculator in React JS by purogrammer

This calculator is fully functional and performs all basic operations.\
Night mode and dark mode is also available.

## Folder Structure

- src (D)

        - Keyboard (D)
            - Keyboard.js
            - Keyboard.css
        - Screen (D)
            - Screen.js
            - Screen.css
        -Calculator.js
    2. App.js
    3. index.js
